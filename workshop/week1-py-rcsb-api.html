What is py-rcsb-api?

py-rcsb-api is a Python toolkit for accessing RCSB PDB services such as the Search API and Data API.

Search API

The Search API helps find PDB entries that match conditions.

Text search

from rcsbapi.search import TextQuery
query = TextQuery(value="Hemoglobin")
results = list(query())
print(results[:10])

Attribute search

from rcsbapi.search import AttributeQuery
query = AttributeQuery(
    attribute="rcsb_entity_source_organism.scientific_name",
    operator="exact_match",
    value="Homo sapiens"
)

Combining queries

q1 = TextQuery(value="Hemoglobin")
q2 = attrs.rcsb_entity_source_organism.scientific_name == "Homo sapiens"
query = q1 & q2

Data API

The Data API retrieves structured information for known PDB IDs.

from rcsbapi.data import DataQuery

query = DataQuery(
    input_type="entries",
    input_ids=["4HHB"],
    return_data_list=["rcsb_id", "struct.title", "exptl.method"]
)

result = query.exec()

Exercises
