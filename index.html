ENMs Workshop

Course notes, reading material, and hands-on exercises prepared as a lightweight static website for GitHub Pages.

Welcome to the workshop website for Elastic Network Models (ENMs). The site is now built from regular HTML pages in the repository root, so it can be hosted directly from GitHub Pages with no Jekyll setup on your machine.

Workshop Guide See the lesson roadmap, references, and site structure. Week 1: Introduction to ENMs Core concepts, collective motions, and spring-based intuition. Week 1: py-rcsb-api Search and data API examples for retrieving PDB information. Bonus Notes Optional theory notes on harmonic potentials and oscillators.

What this site includes

  • A browser-friendly workshop homepage.
  • Static lesson pages served directly by GitHub Pages.
  • Support for the existing images inside images/.
  • Links back to the original markdown lesson files for editing.

Repository workflow

You can keep editing the lesson notes in markdown under workshop/ and update the matching HTML pages when you want the public website to change.

Publishing on GitHub Pages

Push the repository to GitHub, then in Settings -> Pages publish from your default branch and the / (root) folder.
